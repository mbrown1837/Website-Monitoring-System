Environment Variable Prefix

All environment variables must be prefixed with WEBSITE_MONITOR_

Example Environment Variables
Environment Variable Description Example Value
WEBSITE_MONITOR_DASHBOARD_PORT Dashboard port number 5001
WEBSITE_MONITOR_LOG_LEVEL Logging level INFO
WEBSITE_MONITOR_SMTP_SERVER SMTP server for email notifications smtp.gmail.com
WEBSITE_MONITOR_SCHEDULER_ENABLED Enable/disable scheduler true
Data Types
  • Boolean: true, false, 1, 0
  • Integer: Numeric values like 5001, 30
  • Float: Decimal values like 0.5, 1.0
  • String: Text values like INFO, smtp.gmail.com
  • List: Comma-separated values like 500,502,503,504
Production Requirements

In production environment, the following variables are required:

  • WEBSITE_MONITOR_SMTP_SERVER
  • WEBSITE_MONITOR_SMTP_USERNAME
  • WEBSITE_MONITOR_SMTP_PASSWORD
